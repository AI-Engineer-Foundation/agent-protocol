Thank you for your interest in contributing to the AI Engineer Foundation and its projects. Your contributions are essential to our mission of advancing AI engineering. Please take a moment to read through this guide to understand how you can contribute effectively.

## Code of Conduct

We expect all contributors to follow our [Code of Conduct](/contributing/code-of-conduct). Please make sure you are familiar with its content, as it outlines the standards of behavior for all participants in our community.

## How Can I Contribute?

There are several ways you can contribute to the AI Engineer Foundation and its projects:

- **Reporting Issues:** If you encounter bugs or have suggestions, please open an issue in the appropriate project repository.

- **Feature Requests:** If you have an idea for a new feature, feel free to open an issue to discuss it.

- **Documentation:** Help us improve our documentation by fixing errors, adding explanations, or suggesting clarifications.

- **Code Contributions:** If you're a developer, you can contribute code by submitting pull requests.

- **Testing:** We appreciate thorough testing and test coverage to ensure the reliability of our projects.

- **Reviewing Pull Requests:** Reviewing and providing feedback on pull requests is a valuable contribution.

## Contributing to Specific Projects

Below is a list of all of the repositories we have.

- [Agent Protocol Main Repository](https://github.com/ai-engineer-foundation/agent-protocol): This repository contains the spec and testing suite.

- [agentprotocol.ai](https://github.com/ai-engineer-foundation/agentprotocol.ai): This repository contains the documentation website (this website).

- [Python SDK](https://github.com/AI-Engineer-Foundation/agent-protocol/tree/main/packages/sdk/python): This is the Python SDK available on PyPi as `agent-protocol`.

- [JavaScript SDK](https://github.com/AI-Engineer-Foundation/agent-protocol/tree/main/packages/sdk/js): This is the JavaScript SDK available on NPM as `agent-protocol`.

- [Python Client](https://github.com/AI-Engineer-Foundation/agent-protocol/tree/main/packages/client/python): This is the Python Client available on PyPi as `agent-protocol-client`.

## Bug Reports and Feature Requests

Before submitting a new issue, please search the existing issues to see if your problem or suggestion has already been discussed. If not, feel free to open a new issue, providing as much detail as possible to help us understand the problem or feature request.

## Pull Requests

When submitting a pull request, please follow these guidelines:

- Fork the repository and create a feature branch for your changes.

- Follow the code style guidelines for the specific project.

- Write clear commit messages and include a description of your changes.

- Ensure that your code is well-documented and includes tests where applicable.

- Submit the pull request, and our team will review it as soon as possible.

## Code Style Guidelines

Each project within the AI Engineer Foundation may have its own code style guidelines. Please refer to the individual project's documentation for details on coding conventions and style.

## License

By contributing to our projects, you agree that your contributions will be licensed under the project's respective open-source license. Read the LICENSE file in each project for more details.

Thank you for contributing to the AI Engineer Foundation and helping us advance AI engineering. We appreciate your support!
